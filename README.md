# tfenrichr
A package containing enrichment tests for binding sites.

Depends: 
    R (>= 3.3),
    TFMPvalue,
    motifmatchr

Imports:
    Biostrings,
    GenomicRanges,
    SummarizedExperiment,
    TFBSTools

Recommends: CrobustaTFs

LinkingTo: Rcpp, RcppArmadillo

SystemRequirements: C++11
# Usage
```
cd tfenrichr
make
```
